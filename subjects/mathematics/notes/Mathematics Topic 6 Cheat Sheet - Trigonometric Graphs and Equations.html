Hand trick

Under 0°·30°·45°·60°·90° write 0 1 2 3 4. Then sin = √n ⁄ 2; cos = the same list reversed; tan = sin ÷ cos.

🔑 The partner rules

0° ≤ x ≤ 360°

Calculator gives the first angle x₁. The second solution is:

sin x = k → 180° − x₁
cos x = k → 360° − x₁
tan x = k → 180° + x₁

Then +360° to any negative answer; keep only values in 0°–360°.

🧭 Method — every time

solving trig equations
  1. Rearrange to sin/cos/tan x = k.
  2. First angle from the positive value of k.
  3. Apply the matching partner rule.
  4. Adjust into range; check both fit.

No solution if |k| > 1 for sin or cos. tan always gives two answers.

✏️ Worked · cos

2 cos x + 1 = 0
cos x = −½ cos⁻¹(½) = 60° −ve → 180 − 60 & 180 + 60 x = 120°, 240°

cos is negative in the 2nd & 3rd quadrants → 120° and 240°.

✏️ Worked · sin

sin x = 0.6
x₁ = sin⁻¹0.6 = 36.9° 180 − 36.9 = 143.1° x = 36.9°, 143.1°

Positive sin → 1st & 2nd quadrants; partner is 180 − x₁.

✏️ Worked · tan

tan x = −1.5
from +1.5: tan⁻¹1.5 = 56.3° 180 − 56.3, 360 − 56.3 x = 123.7°, 303.7°

Negative tan → 2nd & 4th quadrants → 123.7° and 303.7°.

💡 Why two answers?

the graph view

"Solve sin x = 0.5" asks where the flat line y = 0.5 cuts the sine wave. It hits twice in 0°–360°: the calculator gives one crossing (30°), the graph's symmetry gives the other (150°).

📌 Memory hooks

lock these in
  • Sine → 180 − x
  • Cos → 360 − x
  • Tan → 180 + x

sin & cos reflect (stuck in −1..1); tan repeats every 180° and reaches every value.

🔁 Different range?

e.g. −180° to 180°

If the interval isn't 0°–360°, find all answers with the partner rule first, then shift each by ±360° until it lands inside the range asked. Keep every value that fits.
